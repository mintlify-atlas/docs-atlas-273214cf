## Contributing to Fontcy

Contributions are welcome. If you would like to add a new font style, fix a bug, or improve the UI, open a pull request on [GitHub](https://github.com/b1naryn1nja/fontcy). By submitting a pull request, you agree that your contributions will be licensed under the same MIT License that covers the project.

If you have a style you would like to see added but prefer not to submit code yourself, you can open an issue at [github.com/B1naryN1nja/Fontcy/issues](https://github.com/B1naryN1nja/Fontcy/issues) to request it.
