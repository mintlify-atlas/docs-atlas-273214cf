Fontcy is a pure browser-based tool that converts plain text into Unicode-styled characters you can paste anywhere — social media bios, Discord names, email subjects, or any field that accepts plain text. It was built out of frustration with ad-cluttered, poorly designed alternatives, and it stays intentionally minimal: one HTML file, no frameworks, no sign-up.

## What is Unicode text styling?

The styled text Fontcy produces is not formatted text — it is made up of real Unicode characters. The Unicode standard includes multiple mathematical and letterlike character ranges that happen to resemble bold, italic, script, and other typographic styles. For example, the bold sans-serif letter "a" is the Unicode character `𝗮` (U+1D5EE), not the letter "a" with CSS applied to it.

Because these are standard text characters, they travel wherever plain text does. A Unicode bold letter copied from Fontcy looks the same in a tweet, a LinkedIn bio, a Slack message, or a PDF — no font installation or markup support required on the receiving end.

## Key features

- **Ad-free, always free** — no paywalls, no account, no upsells
- **Open source** — MIT licensed; fork it, mod it, or self-host it
- **18 curated styles** — only styles that render correctly and look intentional are included
- **Zero dependencies** — plain HTML, CSS, and JavaScript; just open `index.html`
- **Click to copy** — one click on any font box copies the styled text to your clipboard
- **Real-time preview** — all 18 styles update as you type, with no delay

<Note>
  Fontcy works entirely in the browser — no account, no server, no data stored.
</Note>
