Fontcy includes 18 Unicode font styles, each of which maps standard Latin letters to a specific Unicode block. When you type into Fontcy, every style is generated instantly in parallel — no selection needed. Click any style box to copy it.

## How styles work

Each style is a 52-character Unicode mapping covering a–z and A–Z. When Fontcy receives your input, it replaces each letter with the corresponding Unicode character from the active style's map. Characters outside the a–z / A–Z range — digits, spaces, punctuation — are left as-is and passed through to the output unchanged.

<Tip>
  Some Unicode characters may not render correctly in all apps. If you see boxes or question marks, try a different style.
</Tip>
