description: "Discover Fontcy's four decorative styles — Small Caps, Fancy 1, Curvy, and Gothic Look — built for creative expression in social bios and posts."
---

Fontcy's decorative styles take a different approach from the mathematical Unicode blocks. Instead of mapping letters to a single contiguous Unicode range, each style draws from multiple sources — modifier letters, Cyrillic lookalikes, Latin Extended characters, and other Unicode substitutions — to produce a distinctive visual effect. Use these styles when you want something more creative or personal than a standard font weight.

<AccordionGroup>
  <Accordion title="Small Caps">
    Small Caps renders lowercase letters as small uppercase characters using Unicode modifier letters (U+1D00–U+1D2F). The result is clean and understated — text looks fully capitalised but at a reduced visual weight.

    **Example:** ʜᴇʟʟᴏ ᴡᴏʀʟᴅ

    **Best for:** Names, headings, and usernames where you want a polished, typographic look. Small Caps pairs well with regular text because it does not look out of place in a sentence.
  </Accordion>

  <Accordion title="Fancy 1">
    Fancy 1 mixes characters from Latin Extended, Cyrillic, and other Unicode blocks to create a stylised alphabet where each letter is swapped for a visually similar but distinctive lookalike.

    **Example:** ȺBCDEℱGHIJKLMNOPQRSƬUVWXYƵ

    **Best for:** Eye-catching usernames, creative bios, and anywhere you want text that looks recognisable but clearly styled. Because it uses multiple Unicode sources, rendering varies slightly across platforms.
  </Accordion>

  <Accordion title="Curvy">
    Curvy uses soft, rounded Unicode characters and Latin Extended lookalikes to give text a flowing, organic feel. The letterforms are smooth and gentle compared to the harder edges of styles like Bold Sans or Squared.

    **Example:** ƈυɾʋყ ʅσσƙʂ ɠɾҽαƚ

    **Best for:** Artistic bios, lifestyle content, and any context where you want text that feels soft and approachable rather than bold and assertive.
  </Accordion>

  <Accordion title="Gothic Look">
    Gothic Look uses per-letter substitutions from Cyrillic, Greek, and other Unicode blocks to approximate a gothic vibe without using full blackletter Fraktur characters. The result is lighter and more casual than Fraktur but still reads as stylised.

    **Example:** thє quíck brσwn fσх

    **Best for:** Casual gothic aesthetics, social media posts, and usernames where you want a gothic feel that is still easy to read. If you need a more faithful blackletter look, use [Fraktur](/fonts/serif-styles) instead.
  </Accordion>
</AccordionGroup>
