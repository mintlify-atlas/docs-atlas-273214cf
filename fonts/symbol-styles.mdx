---
title: "Bubble, Squared, and fullwidth: symbol styles in Fontcy"
description: "Learn how Fontcy's five enclosed styles — Bubble, Squared, Negative variants, and Wide/Fullwidth — wrap letters in geometric shapes for bold visual impact."
---

Fontcy's symbol and enclosed styles wrap each letter inside a geometric shape — a circle, a filled circle, a square, or a filled square — or expand it to full-width. These styles are among the most visually distinctive in Fontcy, making them a popular choice for headings, labels, and decorative lists. Each shape maps to a dedicated Unicode block designed specifically for enclosed alphanumeric characters.

<Info>
  Enclosed styles (Bubble, Squared, and their negative variants) only cover letters. Numbers and symbols remain unchanged.
</Info>

<AccordionGroup>
  <Accordion title="Bubble">
    Bubble wraps each letter in a circle using Unicode's Enclosed Alphanumerics block (U+24B6–U+24E9). Uppercase letters use outlined circles; lowercase letters use smaller circles.

    **Example:** Ⓗⓔⓛⓛⓞ ⓦⓞⓡⓛⓓ

    **Best for:** Playful, friendly content — social posts, decorative lists, and anywhere you want a light, rounded aesthetic.
  </Accordion>

  <Accordion title="Negative Bubble">
    Negative Bubble places letters inside solid (filled) circles. The filled background creates a strong contrast between the letter and its container.

    **Example:** 🅗🅔🅛🅛🅞

    **Best for:** Bold labels, step indicators, and anywhere you want each letter to stand out individually. The high contrast makes this style particularly readable even at small sizes.
  </Accordion>

  <Accordion title="Squared">
    Squared wraps each letter in an outlined square using Unicode's Enclosed Alphanumeric Supplement block (U+1F130–U+1F149). The result is clean and structured — like a keyboard key or a button label.

    **Example:** 🄷🄴🄻🄻🄾

    **Best for:** Modern, grid-aligned aesthetics. Works well for usernames, titles, and any content where you want a clean, geometric look.
  </Accordion>

  <Accordion title="Negative Square">
    Negative Square fills the square with a solid background, inverting the contrast so the letter appears light against a dark square. This is the highest-contrast enclosed style in Fontcy.

    **Example:** 🅷🅴🅻🅻🅾

    **Best for:** Maximum visual weight. Use this style for headings, emphasis, or any situation where you want the text to command attention immediately.
  </Accordion>

  <Accordion title="Wide / Fullwidth">
    Wide/Fullwidth maps each letter to its full-width Unicode equivalent (U+FF21–U+FF5A). Full-width characters were designed for East Asian text contexts where Latin letters need to occupy the same cell width as CJK characters.

    **Example:** Ｈｅｌｌｏ　Ｗｏｒｌｄ

    **Best for:** Aesthetic text with a spacious, vaporwave, or East Asian typography feel. Each character is approximately twice as wide as its standard counterpart, so short words work better than long ones.
  </Accordion>
</AccordionGroup>

<Tip>
  Negative (filled) variants are great for headings and emphasis where you want maximum visual weight.
</Tip>
